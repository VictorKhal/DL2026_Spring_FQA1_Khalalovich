Выбор пал на Polymer Shop неслучайно: его архитектура на базе Polymer и активное использование Shadow DOM делают его идеальным полигоном для отработки навыков автоматизации в условиях, приближенных к современным фронтенд-реалиям. Это позволяет проверить не только базовые сценарии, но и корректность взаимодействия тестов с теневой моделью документа.
